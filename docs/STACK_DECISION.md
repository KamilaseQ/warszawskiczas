# Stack decision

## Final recommendation for website MVP
Use:
- Next.js App Router
- TypeScript
- Tailwind CSS
- custom premium UI system
- Hostinger Node.js Web App hosting
- monorepo
- one website app containing public site + early internal offer-management area

## Why
This gives:
- strong control over premium design
- strong control over SEO and metadata
- a clean route to later internal functionality
- better long-term flexibility than a builder-led setup
- less fragmentation than separate systems too early

## Explicitly postponed
- owner PWA
- large analytics module
- IG/FB integrations
- advanced comms hub
- full CMS complexity
