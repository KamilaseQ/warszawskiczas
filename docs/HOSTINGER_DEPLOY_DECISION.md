# Hostinger deployment decision

## Use
Hostinger Node.js Web App hosting for the Next.js website.

## Do not use as final architecture
Hostinger Website Builder for the core site implementation.

## Deployment shape
- Git-based deployment
- one website app target
- environment variables defined explicitly
- custom domain connected after validation

## Validation after deploy
- homepage loads
- metadata visible
- key pages render
- forms submit correctly
- email delivery works
- domain and SSL work
- no broken internal navigation
