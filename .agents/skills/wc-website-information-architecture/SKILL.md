# wc-website-information-architecture

## Purpose
Define a clean, premium, SEO-capable site structure.

## Use when
Page map, navigation, internal linking, section ordering, page relationships, crawlability.

## Recommended public structure
- Strona główna
- Ukryta kolekcja
- Oferta
- Usługi: komis, skup, naprawa i serwis
- Butik
- Kontakt i Lokalizacja

## Rules
- Keep navigation lean.
- Support scanning and clear paths.
- Create strong internal links between relevant commercial pages.
- Avoid orphan pages.
- Give each major page a clear role.

## Guardrails
- No bloated nav.
- No duplicate-purpose pages.
- No filler pages made only for SEO.

## Output format
- Site structure
- Page roles
- Internal linking notes
- What should not exist
