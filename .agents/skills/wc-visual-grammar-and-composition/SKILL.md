# wc-visual-grammar-and-composition

## Purpose
Define the visual grammar of the Warszawski Czas website: composition, spacing, rhythm, image usage, section pacing, visual tension, CTA placement, and the overall premium editorial feel.

## Use when
Use for homepage layout, page composition, section ordering, visual hierarchy, image/text balance, CTA placement, spacing decisions, grid decisions, and refinements of public-facing page design.

## Do not use when
Do not use for backend, data modeling, internal tools, admin workflows, or purely technical implementation tasks unrelated to visual composition.

## Project truths
- Warszawski Czas must feel like a luxury boutique, not a marketplace or SaaS product.
- The visual system must communicate prestige, restraint, confidence, and curation.
- The site should feel premium and memorable without becoming loud or theatrical.
- Design must support lead generation without looking like an aggressive conversion funnel.
- Dark green and gold are secondary accent colors; the base visual system should remain elegant and restrained.
- Layout clarity and scanability are mandatory.

## Core visual philosophy
- Use editorial composition rather than repetitive box-based UI.
- Let spacing create status and calm.
- Use asymmetry but carefully, to create sophistication, not chaos.
- Build desire through restraint, not overload.
- Use strong visual accents in right order to keep pagescanning flow
- Every page should breathe.
- Premium design should feel controlled

## Composition rules
- Avoid stacking too many sections with the same structure.
- Alternate visual density: not every section should carry the same weight.
- Use a clear rhythm of expansion and compression across the page.
- Use image-led sections strategically, not constantly.
- Maintain consistent alignment logic even when using asymmetry.
- Let important sections feel anchored and composed, not floating.

## Spacing rules
- Use generous whitespace as a sign of premium quality.
- Increase spacing around key transitions and major emotional moments.
- Avoid cramped cards, crowded text columns, and overly dense grids.
- Maintain strong vertical rhythm from section to section.
- Reduce noise before adding decorative elements.
- If a page feels busy, simplify spacing and structure first.

## Hierarchy rules
- The hierarchy must be obvious.
- Headings should feel premium and intentional, never generic.
- Supporting text should remain concise and controlled.
- Primary CTA must be clear
- Secondary actions should support exploration, not compete with primary action.
- Product or service emphasis should never overwhelm the brand tone.

## Image and media rules
- Images must feel curated, premium, and emotionally aligned with the page intent.
- Prefer fewer, stronger images over many weak ones.
- Use media to create atmosphere, scale, materiality, and desire.
- Avoid images that feel too commercial, too busy, or too generic.
- Placeholder media should still feel elegant and on-brand.
- Video areas can use refined poster-style placeholders if real video is not ready.

## Section rhythm rules
- A page should not feel flat from top to bottom.
- Start strong, vary pace in the middle, and close with controlled confidence.
- Use contrast between sections:
  - expressive vs restrained
  - image-led vs text-led
  - emotional vs informative
- Do not let every section scream for attention.
- Some sections should persuade quietly.

## CTA rules
- CTA placement must feel earned, not forced.
- CTA should appear frequently to minimize friction, but it must fit well into the page layout.
- Homepage may carry stronger CTA presence than deeper pages.
- Boutique, contact, and hidden collection pages should use more refined CTA energy.
- Avoid repeating the exact same CTA block style too many times.
- CTA language should feel premium, calm, and natural.
- The page should guide the visitor

## Grid and layout rules
- Use structured layouts with enough flexibility for editorial rhythm.
- Avoid rigid sameness in every section.
- Avoid over-fragmented small cards.
- Prefer broader blocks and more confident composition.
- Product lists should remain clean and premium, not cluttered.
- Service pages may be slightly more structured, but still refined.

## Visual tension rules
- Tension should come from composition, scale, contrast, and restraint.
- Do not rely on gold, shadows, or dark backgrounds to fake prestige.
- Use subtle contrast, spacing, and image quality to create perceived value.
- If something feels too “designed,” simplify it.

## Mobile composition rules
- Mobile must preserve hierarchy and elegance.
- Do not simply stack desktop ideas thoughtlessly.
- Reduce visual clutter aggressively on mobile.
- Keep hero impact while protecting readability.
- Maintain clear spacing between sections.
- Do not overload mobile screens with too many competing elements above the fold.
- The mobile version may differ from the desktop version if it improves the composition

## Per-page visual intensity guidance
- Homepage: highest visual intensity, strongest editorial rhythm, flagship expression.
- Offer: calmer, more ordered, more selective, more catalog-driven.
- Hidden collection: more restrained, more suggestive, more tension through space, more misterious, more secret.
- Services: more structured, more trust-driven, less theatrical.
- Boutique: more atmospheric, place-driven, experience-led.
- Contact: simpler, elegant, highly legible, premium concierge tone.

## Anti-patterns
- Generic ecommerce grids with no atmosphere
- Repetitive card-on-card layouts
- Overuse of gold accents
- Dark luxury clichés
- Oversized rounded components with soft startup aesthetics
- Loud gradients and fake-premium effects
- Too many visual ideas competing in one section
- Every section trying to be the hero
- Design choices that reduce scanability

## Required questions before designing a section
- What role does this section play in the page rhythm?
- Should this section expand the experience, clarify something, or convert attention?
- Does this section need visual intensity, or should it calm the page down?
- Is this section image-led, text-led, or balance-led?
- Does this help the page feel more premium, or just more busy?

## Guardrails
- Do not design every section with equal visual weight.
- Do not use decoration as a substitute for composition.
- Do not increase complexity when spacing and hierarchy are the real issue.
- Do not let premium become cold, empty, or lifeless.
- Do not let “luxury” become cliché.

## Checklist
- Is the composition elegant and controlled?
- Is the hierarchy instantly readable?
- Does the page have rhythm?
- Are spacing and media doing enough work?
- Are CTAs placed naturally?
- Does the layout feel premium instead of generic?
- Is mobile still strong and uncluttered?

## Output format
