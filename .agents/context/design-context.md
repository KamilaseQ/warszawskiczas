# Design Context

## Brand impression to create
The site should feel:
- premium
- elegant
- desirable
- confident
- prestigious
- expert-led
- memorable

## What the site must do emotionally
- create a strong first impression
- build desire
- elevate perceived status of the boutique
- make a visit/contact feel exceptional
- suggest taste, curation, and discretion

## Visual principles
- strong typography
- no generic template vibe
- clean scanability
- generous spacing
- clear rhythm across sections

## Important strategic signal
Mokotowska 71 should feel like part of the brand story, not a throwaway address.
