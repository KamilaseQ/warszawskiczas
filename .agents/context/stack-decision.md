# Stack Decision Context

## Chosen stack for website
- Framework: Next.js (App Router)
- Language: TypeScript
- Styling: Tailwind CSS
- UI primitives: shadcn/ui where useful, but heavily customized
- Forms: server actions or route handlers + transactional email handling
- Hosting: Hostinger Node.js Web App hosting
- Repository shape: monorepo
- Data ownership: website app owns public site + early offer management
- Future app: separate owner PWA later

## Why this stack is chosen
- Strong fit for premium custom website work
- Good SEO and routing control
- Strong performance potential
- Easy to structure public pages and secure internal admin pages
- Good path to future expansion without overbuilding now

## Explicit non-choice
- Do not use Hostinger Website Builder for the final website architecture
- Do not split the site into separate marketing frontend and separate CMS frontend at this stage
- Do not create the owner PWA now
